# Research-paper

## Overview

This repository contains the research paper:

**Decoding Brain Tumors: An Attention-Guided Hybrid Learning Framework for Multi-Class Classification of MRI Images**

The work presents an attention-guided hybrid deep learning framework for the automated multi-class classification of brain tumors using MRI images. The proposed approach aims to improve classification accuracy by combining feature extraction, attention mechanisms, and deep learning architectures for robust medical image analysis.

## Abstract

Brain tumor diagnosis from MRI scans is a challenging task due to the complex structure and visual similarity among different tumor types. This research proposes an attention-guided hybrid learning framework that enhances feature representation and classification performance for multi-class brain tumor recognition. The proposed model leverages deep learning techniques along with attention mechanisms to improve diagnostic accuracy while maintaining computational efficiency.
